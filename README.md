# JavaScript & TypeScript Exercises

A collection of coding exercises and small assignments covering fundamental programming concepts in JavaScript (and later TypeScript).

## Topics Covered

- Variables and data types
- Functions and parameters
- Arrays and loops
- Conditionals and basic logic
- String manipulation
- Introductory DOM work
- General problem-solving patterns

## Purpose

The goal of this repository is to build a solid foundation in core programming concepts through repeated practice.
