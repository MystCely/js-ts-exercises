# JavaScript & TypeScript Exercises

A practice workspace of small front-end exercises built with HTML, CSS, and JavaScript.

## What This Project Is

A learning-focused repository with:

- foundational JavaScript drills
- DOM interaction mini-projects
- small MVC-style exercises

## Features

- Multiple standalone exercises in one repo
- Mix of beginner logic practice (loops, variables, strings, functions)
- Interactive browser mini-projects (e.g., stopwatch, traffic light, validation, games)
- Some projects organized with `model/view/controller` separation
- Local assets included per exercise (`img/` folders)

## Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## Usage Notes

- Each exercise is self-contained in its own folder.
- Open any `index.html` file directly in your browser to run that exercise.
- Projects with a `script/` directory typically separate logic by responsibility (`model`, `view`, `controller`)
