# JavaScript & TypeScript Exercises

A practice workspace with small front-end exercises built during learning sessions.

## What Is In This Repo

- Standalone JavaScript exercises (logic drills + DOM mini-projects)
- TypeScript mini-apps built with Vite
- A mix of simple scripts and projects with separated logic/UI modules

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- TypeScript
- Vite

## TypeScript Apps

- `contact-list`: add, edit, and delete contacts
- `todo-list`: add todos, mark complete, and delete
- `shopping-list`: filter items, toggle bought state, and remove items
- `book-collection`: add books, mark read, and delete entries
